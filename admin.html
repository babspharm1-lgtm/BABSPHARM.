📦 Bulk Upload Questions
Upload many questions at once. Supports CSV, JSON, TXT or any text file.
CSV Format: Download template below. Each row = 1 question.
Columns: question, optionA, optionB, optionC, optionD, answer
Answer = A, B, C, or D

JSON Format:
[{"q":"Question?","opts":["A","B","C","D"],"ans":0}]
ans = 0 for A, 1 for B, 2 for C, 3 for D
Text Format: Paste questions in this format:
1. What is...? A. Option 1 B. Option 2 C. Option 3 D. Option 4 Answer: A
